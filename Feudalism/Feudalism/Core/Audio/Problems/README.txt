Audio snimke problema – hrvatski (hr) i engleski (en)
====================================================

U mapi hr/ stavi snimke na HRVATSKOM, u mapi en/ snimke na ENGLESKOM.
Format: .wav ili .mp3. Ime datoteke = ključ (bez razmaka, mala slova).

Lista datoteka (iste nazive napravi u obje mape hr/ i en/):

  audio_problem_file_not_found
    hr: "Zvučna datoteka nije pronađena."
    en: "Audio file not found."

  audio_problem_playback_failed
    hr: "Reprodukcija nije uspjela."
    en: "Playback failed."

  audio_problem_no_device
    hr: "Nema dostupnog izlaza za zvuk."
    en: "No audio output device available."

  audio_problem_volume_muted
    hr: "Glasnoća je isključena."
    en: "Volume is muted."

  audio_problem_initialization_failed
    hr: "Inicijalizacija zvuka nije uspjela."
    en: "Audio initialization failed."

  audio_problem_speech_unavailable
    hr: "Sintetizator govora nije dostupan."
    en: "Speech synthesizer is not available."

Primjer:
  hr/audio_problem_file_not_found.wav   <- snimka na hrvatskom
  en/audio_problem_file_not_found.wav  <- snimka na engleskom
